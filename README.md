Material de clase del curso **Python + IA aplicado a la Geoinformación**: una introducción práctica al uso de Python, librerías geoespaciales y técnicas de Inteligencia Artificial para el análisis de información geográfica, con ejemplos aplicados a Colombia.

## Contenido del repositorio

| Recurso | Descripción |
|---|---|
| [`presentacion/`](presentacion/sesion1_conceptos.html) | Diapositivas web de la sesión 1 (también disponibles en el enlace de arriba) |
| [`clase 1/sesion1.ipynb`](clase%201/sesion1.ipynb) | Notebook: introducción a GeoPandas y manejo de shapefiles (departamentos de Colombia) |
| [`clase 2/sesion_2.ipynb`](clase%202/sesion_2.ipynb) | Notebook: análisis con datos de estaciones IDEAM y capas departamentales |
| `curso_python+IA_geoinformacion.pdf` | Documento del curso en PDF |

## Temas que se abordan

- Fundamentos de Sistemas de Información Geográfica (SIG)
- Datos vector vs ráster, capas, coordenadas y proyecciones
- Uso de Python en Google Colab para análisis geoespacial
- Librerías: `geopandas`, manejo de shapefiles
- Conexión entre Python, IA y geoinformación

## Cómo usar este material

1. Revisa la <a href="https://trabajocientifico.github.io/curso-python-IA-geoinformacion/" target="_blank" rel="noopener noreferrer">presentación en línea</a> para los conceptos.
2. Abre los notebooks (`.ipynb`) en [Google Colab](https://colab.research.google.com/) o Jupyter.
3. Los datos de ejemplo están dentro de cada carpeta `datos/`.

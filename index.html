¿Qué es un SIG?

Un Sistema de Información Geográfica captura, almacena, analiza y visualiza datos con ubicación en la Tierra.

80%
de datos tienen componente geoespacial

¿Dónde?

Todo geodato responde "¿dónde está?". Hospital, río, finca — todos tienen coordenadas.

📊

¿Qué hay ahí?

Además de ubicación: nombre, tipo de suelo, temperatura, uso del terreno.

🔍

¿Qué patrones?

Descubrir relaciones espaciales: ¿La deforestación sigue a las carreteras?

Análisis multitemporal: comparar un lugar en diferentes momentos.

sistemas de referencia

Coordenadas y Proyecciones

Toda geoinformación necesita un sistema de referencia. Haz clic en el mapa:

📍 Latitud (Norte-Sur)

4.1420°

📍 Longitud (Este-Oeste)

-73.6266°
# En Python: latitud = 4.1420 longitud = -73.6266 punto = [latitud, longitud]
CRS: El más común es WGS84 (EPSG:4326) del GPS. Colombia usa MAGNA-SIRGAS. GeoPandas maneja proyecciones automáticamente.

Python como calculadora geoespacial

Modifica coordenadas y ve la distancia en tiempo real:

Calculadora de distancia
DISTANCIA APROXIMADA

distancia.py
# Distancia simplificada import math lat1, lon1 = 4.142, -73.626 lat2, lon2 = 4.601, -74.072 km = math.sqrt((lat2-lat1)**2+(lon2-lon1)**2)*111.32 print(f"📏 Distancia: {km:.1f} km")
Salida
Operadores: + - * / ** (potencia) math.sqrt() (raíz)
